# Collection cover images

Place your cover art in this folder using these exact filenames:

- `raja-ampat.jpg`
- `komodo.jpg`
- `aerials.jpg`
- `lembeh-strait.jpg`
- `megafauna.jpg`
- `culture.jpg`

Recommended size: 800×450px or larger (16:9). JPG or WebP.

You may also copy assets from a project **Covers** folder into `images/covers/` using the names above.
